This project performs Exploratory Data Analysis (EDA) on a Iris dataset.

The EDA includes:

Data Cleaning

Checked for missing values and cleaned the dataset.
Dropped rows with missing values in critical columns to ensure accurate analysis.
Cleaned column names to remove extra spaces or inconsistencies.
Summary Statistics

Mean, Median, Mode, Standard Deviation
Helps understand central tendency and variability of stock prices
Data Visualization

Histograms, Boxplots, Scatter plots
Understand distributions, outliers, and relationships between features
Correlation Analysis


Key Learning Goals
Understand and calculate summary statistics for all numerical columns
Learn how to visualize data distributions and identify patterns
Discover correlations between features to guide further analysis
Practice step-by-step Python data analysis workflow
Learn proper data cleaning techniques before analysis
